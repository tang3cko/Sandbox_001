# Rain

## Purpose

This experiment explores SpotLight parameter synchronization to GPU via VariableSO. The primary goal is to learn rim lighting techniques where rain particles glow when illuminated by light sources.

---

## Learning goals

- SpotLight parameter synchronization using `VariableSO`
- GPU Sync pattern with `Tang3cko.ReactiveSO`
- Rim lighting implementation for particles
- Light-particle interaction in shaders

---

## Dependencies

- `Tang3cko.ReactiveSO` package

---

## Scripts

| Script | Description |
|--------|-------------|
| `SpotLightSync.cs` | Syncs SpotLight parameters to VariableSO assets |

---

## Future extensions

- PointLightSync for point light support
- DirectionalLightSync for directional light support
